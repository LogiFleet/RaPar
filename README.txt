
Command line parameters description
===================================

-man	Manufacturer
-dev	Device

-scp	secure copy (remote file)

-usr	scp: User
-hst	scp: Host
-prt	scp: Port number
-key	scp: RSA public key file path and file name
-pwd	scp: RSA public key file password, nothing if none
-rfd	scp: Remote folder
-rfl	scp: Remote file
-lfd	scp: Local folder

-ts		Time stamp in log
-rd		Raw data in output
